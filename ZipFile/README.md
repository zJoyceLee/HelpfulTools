# Zip File
Zip all direct folder in current path

### Directory Structure
```bash
.
├───aa
│   └───aaa
│   └───aa.md
│   └───aa.txt
├───bb
│   └───bb.md
├───cc
│   └───cc.jpg
├───foo.txt
└───main.py
```
### How to run?
```cmd
python main.py
```

### How to use?
Copy main.py to the dir which you want to zip all direct folders.


### Result Structure
```bash
.
├───aa
│   └───aaa
│   └───aa.md
│   └───aa.txt
├───bb
│   └───bb.md
├───cc
│   └───cc.jpg
├───aa.zip
├───bb.zip
├───cc.zip
├───foo.txt
└───main.py
```
